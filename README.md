#ui-markdown
Polymer element which loads inline or referenced markdown.

## Getting Started

1.  Install bower(do once)

  ```
  $ npm install -g bower
  ```

1.  Initialize the element

  ```
  $ npm install
  $ npm test
  ```

1.  Navigate your browser to `localhost:10000`

## API
<ui-markdown ></ui-markdown>
